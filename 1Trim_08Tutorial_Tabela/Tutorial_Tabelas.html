O que é uma tabela?

As tabelas são listas de dados em duas dimensões e são compostas por linhas e colunas.
Portanto, são muito utilizadas para apresentar dados de uma forma organizada.

Tags Estruturais de uma Tabela

As tags e suas funções utilizadas para a estrutura das tabelas são:

  • < table border="1"> e < /table >: Criar tabela com borda
  • < table > e < /table >: Criar tabela sem borda
  • < tr > e < /tr >: Criar cada linha
  • < td > e < /td >: Criar os elementos
  • < th > e < /th >: Representar Título
  • < thead > e < /thead >: Cabeçalho, geralmente título
  • < tbody > e < /tbody >: Corpo da tabela
  • < tfoot > e < /tfoot >: Rodapé da tabela
  • < caption > e < /caption >: Título da tabela
  • < colgroup >< /colgroup > e < col class="">< /col >: Direcionar cor vinda do Style

Tabela com Célula de Título

Caso queira incluir uma ou mais células que representem títulos, ganhando um destaque em relação às demais células, pode utilizar a tag < th >.
Vejamos a seguir:

Estrutura

< table border="1"> 
        < tr>
            < th> País < /td>
            < th> Cidade < /td>
            < th> Paisagem  < /td>
        < /tr>
        < tr> 
            < td> Grécia < /td>
            < td> Santorini < /td>
            < td> < a href="img/santorini.jpg" target="_blank"> 
                < img src="img/santorini.jpg"alt="" width="20%"> < /a> < /td>
        < /tr>
        < tr> 
            < td> Emirados Árabes Unidos < /td>
            < td> Dubai  < td>
            < td> < a href="img/Dubai.jpg" target="_blank"> 
                < img src="img/Dubai.jpg" alt="" width="15%"> < /a> < /td>
        < /tr>
< /table>

Resultado

País Cidade Paisagem
Grécia Santorini
Emirados Árabes Unidos Dubai

Tabela Estruturada

Anteriormente, vimos que a estrutura básica de uma tabela consiste nas tags < tr >, < td > e as células de título < th >.
Porém, além dessas, também há tags que servem para estruturar melhor a tabela. São elas:

< thead > e < /thead >
– Esta tag representa o cabeçalho da tabela, geralmente composta por células título.
< tbody > e < /tbody >
– Essa tag representa o corpo da tabela.
< tfoot > e < /tfoot >
– Essa tag representa o rodapé da tabela.

Portanto, essas tags tem como principal efeito manter uma semântica adequada para suas tabelas.
Sendo assim, vejamos então a seguinte imagem para exemplificar a estrutura de uma tabela:

Estrutura

 < table border="1">
        < thead bgcolor="DarkSlateBlue">
            < tr>
                < th> Matéria< /th>
                < th> Aula < /th>
                < th> Exercício < /th>
            < /tr>
        < /thead>
        < tbody bgcolor="Gainsboro">
            < tr>
                < td> DWeb < /td>
                < td> Aula 01 < /td>
                < td> < address> < a href= "Aula01_01.html" target="_blank"> Ex01 - Tags de Texto < /a> < /address> < /td>
            < /tr>
            < tr>
                < td> DWeb < /td>
                < td> Aula 01 < /td>
                < td> < address> < a href= "Aula01_02.html" target="_blank"> Ex02 - Letras de Música < /a> < /address>< /td>
            < /tr>
        < /tbody>
        < tfoot  bgcolor="Navy">
                < td>...< /td>
                < td>...< /td>
                < td>...< /td>
        < tfoot>
< /table>

Resultado

Matéria Aula Exercício
DWeb Aula 01
Ex01 - Tags de Texto
DWeb Aula 01
Ex02 - Letras de Música
... ... ...

Tabela com Célula Vazia

Caso seja necessário incluir uma célula vazia (ou seja: sem nenhum valor dentro) na sua tabela, uma saída seria apenas incluir a < td > ou < th > sem nenhum conteúdo.
No entanto, isso não é recomendado, porque alguns navegadores não conseguem ler adequadamente uma célula vazia.

Estrutura

 < table border="1">
        < tr>

        < /tr>
        < tr>
            < td> Telefones < /td>
            < td> < ul> 
                < li> (42)9 99145-0707 < /li>

            < /ul> < /td>
        < /tr>
        < tr>
            < td> Emails < /td>
            < td> < ul> 
                < li> < a href="mailto:contato@contato.com"> analivialazz@gmail.com < a> < / li>
                < li> < a href="mailto:contato@contato.com"> ana.lazzari.ifpr@gmail.com < /a>  < /li>
            < /ul> < /td>
        < /tr>
< /table>

  • (42)3667-1266

Tabela com Célula mais Abrangente

Em algumas situações, temos a necessidade de utilizar uma mesclagem de célula, ou seja, criar uma célula que abrange mais do que uma linha ou coluna.
Dessa forma, podemos utilizar os atributos colspan e rowspan.
O colspan=”x” vai fazer uma mesclagem de colunas, e o rowspan=”x” vai mesclar linhas, bastando substituir o x pelo número de colunas ou linhas que deseja ocupar.

Título da Tabela

A tabela HTML tem uma tag particular para definir um título. Essa tag é a < caption > .
Em contraste com os demais elementos da tabela, a tag < caption > não será composta por células, portanto ficará acima da tabela, como um título.

Estrutura

< table border="1">
        < caption> < b> HTML < /b> < /caption>
        < tr>
            < th bgcolor="Gainsboro"> Site < /th>
            < th bgcolor="Gainsboro"> Link < /th>
        < /tr>
        < tr>
            < td> W3schools < /td>
            < td> < a href="https://www.w3schools.com/" target="_blank"> W3schools < /a> < /td>
        < /tr>
        < tr>
            < td> Mozilla < /td>
            < td> < a href="https://developer.mozilla.org/pt-BR/" target="_blank"> Mozilla < /a> < /td>
        < /tr>
< /table>

Resultado

HTML
Site Link
W3schools W3schools
Mozilla Mozilla

Utilizando Colgroup e Col

Anteriormente, explicamos que hoje em dia, boa parte dos atributos de uma tabela se encontra depreciado. Portanto, a estilização deve ser feita através do CSS.
Dessa forma, existe um elemento muito importante que nos permite criar um grupo de colunas, podendo assim estilizar diversas colunas de uma só vez.
Esse elemento é a tag < colgroup > e < colgroup >. Posteriormente, dentro do grupo de colunas, incluímos então as tags < col > < /col > para poder definir cada coluna.

Estrutura

(style)

< table border="1">
    < caption> < b> Tags de Formatação < /b> < /caption>
        < colgroup>
            < col class="azul">
            < col class="vermelho">
            < col class="amarelo">
        < /colgroup>
        < tr>
            < th> Tag < /th>
            < th> Código < /th>
            < th> Resultado < /th>
        < /tr>
        < tr>
            < td> del < /td>
            < td> < del> < /del> < /td>
            < td> Texto < del> excluído < /del>! < /td>
        < /tr>
        < tr>
            < td> ins < /td>
            < td> < ins> < /ins> < /td>
            < td> Texto < ins> inserido < /ins>! < /td>
        < /tr>
        < tr>
            < td> sub < /td>
            < td> < sub> < /sub> < /td>
            < td> Texto < sub> subscript < /sub> < /td>
        < /tr>
        < tr>
            < td> sup < /td>
            < td> < sup> < /sup> < /td>
            < td> Texto < sup> superscript < /sup> < /td>
        < /tr>
< /table>

Resultado

Tags de Formatação
Tag Código Resultado
del < del> < /del> Texto excluído!
ins < ins> < /ins> Texto inserido!
sub < sub> < /sub> Texto subscript
sup < sup> < /sup> Texto superscript
