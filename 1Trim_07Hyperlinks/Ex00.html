Hyperlinks

Hyperlinks são referências dentro de um documento em hipertextos para outras partes do mesmo documento ou outros documentos.

Os links podem ser:

  • Remotos
  • Locais
  • Internos
  • Endereço de correio eletrônico
  • Arquivo para download

Hyperlinks Remotos

São hyperlinks que ligam para outros documentos disponíveis na web. Exemplo: Clique aqui!

Hyperlinks Locais

São hyperlinks hospedados em um mesmo servidor. Exemplo: Hyperlink Local

Hyperlinks em Imagens

É possível deixar uma imagem linkada direcionando para uma outra referência.

Hyperlinks Internos

São hyperlinks que referenciam itens dentro do mesmo documento. Exemplo:

  • Capítulo 1
  • Capítulo 2
  • Capítulo 3
