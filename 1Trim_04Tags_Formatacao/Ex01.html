Tags e Aplicações

del

A tag del define o texto que foi excluído de um documento. Exemplos:

Minha cor favorita é azul!
O carro era preto!
del é uma tag que define o texto excluído!

ins

A tag ins define um texto que foi inserido em um documento.

Minha cor favorita é azul red!
O carro era preto cinza!
ins é uma tag que define o texto excluído adicionado!

sub

A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo do linha normal, e às vezes é renderizada em uma fonte menor.

Pode ser usado para fórmulas químicas, como H2O.
sub é a tag para um texto subscript.
H2O é a formula da água.

sup

A tag sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima do linha normal.

O texto contem superscript.
O resultado de 22 é 4.
O formato 22 apresentado acima é uma potenciação.

blockquote

A tag blockquote especifica uma seção que é citada de outra fonte.

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization,
WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
A tag especifica uma seção que é citada de outra fonte.

q

A tag q define uma citação curta.

O objetivo do WWF é: Construa um futuro onde as pessoas vivam em harmonia com a natureza. Esperamos que eles tenham sucesso.
O lema mais conhecido de Sócrates é: Só sei que nada sei.
Como diz Confúcio: Se queres prever o futuro, estuda o passado.

cite

A tag cite define o título de um trabalho criativo, por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.

The Scream by Edward Munch. Painted in 1893.
Iracema by José de Alencar. Desede 1865.

A tag kbd é usada para definir a entrada do teclado. O conteúdo dentro é exibido na fonte monospace padrão do navegador.
