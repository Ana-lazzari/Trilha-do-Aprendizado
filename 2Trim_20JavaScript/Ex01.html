Caixas de Diálogo em JS

São três tipo de caixas de diálogo:

  • alert: exibe uma mensagem de alerta ao usuário
  • prompt: exibe uma mensagem de confirmação em que o usuário pode escolher entre OK e CANCELAR
  • confirm: o usuário pode adicionar dados na caixa de diálogo

É importante destacar que o Java Script pode ser incorporado ao documento HTML pela tag script.
O script js pode ser interno, ou seja, dentro do documento HTML ou externo, no qual será um arquivo externo js.
