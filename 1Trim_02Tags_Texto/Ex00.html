Desenvolvimento Web

Tecnologias utilizadas

Front-End

Front-End é o serviço responsável por desenvolver a interface gráfica do usuário de um site, aplicativo ou software.
Por meio do uso de HTML, CSS e JavaScript para que os usuários possam visualizar e interagir com esse site.

Back-End

Back-End é o o desenvolvimento de recursos e estrutura por trás da programação, que garante a realização das ações
por parte dos usuários em um computador.
Ou seja, resposável pela parte de programação dos códigos.

FullStasck

FullStasck é o responsável por ambas partes, tanto Front-End, como Back-End.
